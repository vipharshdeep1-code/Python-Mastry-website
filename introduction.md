# Introduction to Python

Python is one of the most popular and versatile programming languages in the world. Created by **Guido van Rossum** and first released in **1991**, Python has grown into a language used by millions of developers for web development, data science, artificial intelligence, automation, and much more.

## Why Learn Python?

- **Easy to Read and Write** – Python's syntax is clean and resembles plain English, making it perfect for beginners.
- **Versatile** – From web apps to machine learning, Python can do it all.
- **Huge Community** – Millions of developers and thousands of libraries are available.
- **In Demand** – Python developers are among the highest-paid in the industry.
- **Cross-Platform** – Python runs on Windows, macOS, Linux, and more.

## What Can Python Do?

Python is used in many areas of computing:

| Domain | Examples |
|--------|---------|
| Web Development | Django, Flask |
| Data Science | Pandas, NumPy, Matplotlib |
| Machine Learning | TensorFlow, PyTorch, scikit-learn |
| Automation | Scripting, web scraping, task automation |
| Game Development | Pygame |
| Desktop Apps | Tkinter, PyQt |

## Your First Python Program

Every programming journey starts with the classic "Hello, World!" program. In Python, it's beautifully simple:

```python
print("Hello, World!")
```

The `print()` function outputs text to the screen. That's it — no semicolons, no curly braces, no boilerplate code.

## Try it Yourself

## Python Versions

There are two major versions of Python:

- **Python 2** – Released in 2000, officially retired in January 2020. You should **not** use Python 2 for new projects.
- **Python 3** – The current and actively maintained version. All lessons in this course use Python 3.
