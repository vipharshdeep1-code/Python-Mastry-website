# 🐍 Python Mastery

**Learn Python programming from scratch** with interactive lessons, code examples, and a built-in Python editor that runs in your browser.

## 🌐 Live Site

Visit the live website: [Python Mastery](https://vipharshdeep1-code.github.io/python-mastery/)

## ✨ Features

- 📚 **14 Interactive Lessons** — From Introduction to Debugging
- 💻 **Built-in Code Editor** — Write and run Python right in your browser
- 🎨 **Beautiful UI** — Clean, modern design with syntax highlighting
- 📱 **Responsive** — Works on desktop, tablet, and mobile
